# edf-telereleve
Programme PHP pour lire le télérelevé

# Install

Clone the project or download the tarball.

Open terminal and execute :

```
$ php composer.phar install --no-dev -o
```

# Usage

In the terminal :

```
$ ./telereleve
```

# Tests


Open terminal and execute :

```
$ php composer.phar install -o
$ ./run-unit
```
